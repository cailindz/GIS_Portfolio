# Assignment: Investigating the DEA Pain Pill Database Using ArcGIS Insights
### *4/2/2023*

The goal of this assignment was to perform analysis on pain pill distribution using ArcGIS insights. Using data from the [pain pill database](https://www.washingtonpost.com/graphics/2019/investigations/dea-pain-pill-database/), I was able to visualize the distribution of pain pills in Mingo County, West Virginia.
